## Run Locally

Clone the Quizify

```bash
  git clone https://github.com/EdgarHdzHdz17/Quizify.git
```

Go to the project directory

```bash
  cd Quizify
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
